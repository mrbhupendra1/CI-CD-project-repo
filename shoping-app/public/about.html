About Aurora Shop

Aurora Shop is a demo store built for CI/CD & monitoring demo. This sample app demonstrates a multi-page UI, CSS animations, and a small backend API to integrate with your Jenkins pipeline, Docker builds and monitoring stack.

Tech stack

  • Node.js + Express
  • Static frontend (HTML/CSS/JS)
  • Dockerized app
  • Prometheus + Grafana for monitoring
